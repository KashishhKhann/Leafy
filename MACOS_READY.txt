  cp .env.example .env
  nano .env  # Add your API keys
  python Leafy.py

PLATFORM-SPECIFIC HANDLING:

Window Switching:
  macOS:   Cmd+Tab (via osascript)
  Windows: Alt+Tab (via pyautogui)
  Linux:   Alt+Tab (via pyautogui)

Screenshots:
  macOS:   screencapture command
  Windows: pyautogui
  Linux:   gnome-screenshot

System Control:
  macOS:   AppleScript (osascript)
  Windows: subprocess shutdown commands
  Linux:   systemctl commands

Text-to-Speech:
  macOS:   nsss (Native Speech Synthesis)
  Windows: sapi5
  Linux:   espeak/festival
